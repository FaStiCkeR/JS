## 🏗️ АРХИТЕКТУРА

### Слои

**Слой 1: Utils** (Базовые утилиты)

- constants.js - константы
- helpers.js - вспомогательные функции

**Слой 2: Core** (Бизнес-логика)

- gameState.js - управление состоянием
- gameLogic.js - логика игры
- stats.js - статистика

**Слой 3: UI** (Отрисовка)

- board.js - доска
- keyboard.js - клавиатура
- modal.js - окна
- theme.js - тема

**Слой 4: Input** (Управление)

- keyboardHandler.js - физическая клавиатура
- uiEvents.js - координатор событий

**Слой 5: Main** (Инициализация)

- main.js - точка входа

### Зависимости

```
main.js (верхний уровень)
   ↓
input/ (обработка)
   ↓
ui/ + core/ (логика и отрисовка)
   ↓
utils/ (базовые функции)
```
