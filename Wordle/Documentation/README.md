## 🎯 Обзор

**Wordle** - это словесная игра, где нужно угадать русское слово из 5 букв за 6 попыток.

### Фичи

✅ **Игровой процесс**

- 6 попыток угадать слово
- 293 различных русских слова
- Цветные подсказки (зелёный/жёлтый/серый)
- Автоматическая проверка при вводе 5 букв

✅ **Интерфейс**

- Интерактивная доска 6×5
- Виртуальная русская клавиатура
- Поддержка физической клавиатуры
- Маппинг English → Russian
- Адаптивный дизайн (работает на мобильных)

✅ **Функциональность**

- 🌓 Переключение светлой/тёмной темы
- 📊 Сохранение статистики (localStorage)
- 🎯 Подсчёт процента побед
- 🔄 Кнопка новой игры

## 🎮 Как играть

1. **Введи первое слово** (5 букв)
    - Используй физическую клавиатуру (переводится автоматически)
    - Или нажимай на кнопки виртуальной клавиатуры
    - При вводе 5 букв проверка происходит автоматически

2. **Посмотри на результаты**
    - 🟩 **Зелёный** - буква на правильном месте (correct)
    - 🟨 **Жёлтый** - буква есть, но на другом месте (present)
    - ⬜ **Серый** - буквы нет в слове (absent)

3. **Повторяй** пока не угадаешь или не закончатся попытки

4. **Посмотри статистику** - нажми на иконку таблицы в header

## 🏛️ Архитектура

### 4-слойная архитектура

```
┌─────────────────────────────────────┐
│      PRESENTATION LAYER (UI)        │  board, keyboard, modal, theme
├─────────────────────────────────────┤
│       CONTROL LAYER (INPUT)         │  uiEvents, keyboardHandler
├─────────────────────────────────────┤
│       BUSINESS LAYER (CORE)         │  gameState, gameLogic, stats
├─────────────────────────────────────┤
│       UTILITY LAYER (UTILS)         │  constants, helpers
└─────────────────────────────────────┘
```

### Принципы

- ✅ **SOLID** - каждый модуль одну задачу
- ✅ **DRY** - не повторяй себя
- ✅ **KISS** - просто и понятно
- ✅ **No Global State** - только const
- ✅ **No Circular Dependencies** - зависимости только вверх

## 📦 Модули

### core/ - Бизнес-логика

**gameState.js** (68 строк)

- Хранит состояние игры
- Управляет текущим угаданием
- Не содержит UI логику

**gameLogic.js** (39 строк)

- Проверяет буквы
- Определяет статусы (correct/present/absent)
- Выбирает случайное слово

**stats.js** (57 строк)

- Работает с localStorage
- Сохраняет статистику
- Вычисляет проценты

### ui/ - Отрисовка

**board.js** (83 строк)

- Создаёт сетку 6×5
- Обновляет ячейки
- Управляет DOM для доски

**keyboard.js** (58 строк)

- Отрисовывает клавиатуру
- Обновляет статусы букв
- Управляет виртуальными кнопками

**modal.js** (44 строк)

- Окно со статистикой
- Кнопка новой игры

**theme.js** (33 строк)

- Переключение темы
- Сохранение в localStorage

### input/ - Обработка ввода

**keyboardHandler.js** (31 строк)

- Слушает физическую клавиатуру
- Преобразует English → Russian
- Поддерживает Backspace, Enter

**uiEvents.js** (155 строк)

- Главный координатор
- Обрабатывает все события
- Управляет игровым процессом

### utils/ - Утилиты

**constants.js** (45 строк)

- Список из 20 слов
- Маппинг букв English↔Russian
- Раскладка клавиатуры
- Конфигурация игры

**helpers.js** (35 строк)

- Вспомогательные функции
- Преобразование данных
- Показ сообщений
