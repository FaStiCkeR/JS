## 📄 Все документы

### [README.md](README.md)

Полный гайд по проекту. Описание всех фич, как запустить, как расширять.

- **Для:** Всех
- **Размер:** ~14KB
- **Время чтения:** 15-20 минут

### [ARCHITECTURE.md](ARCHITECTURE.md)

Описание архитектуры проекта. 4-слойная система, принципы, паттерны.

- **Для:** Архитекторов и старших разработчиков
- **Размер:** ~6KB
- **Время чтения:** 10-15 минут

### [API.md](API.md)

Полный список API каждого модуля. Какие функции экспортирует каждый файл.

- **Для:** Разработчиков
- **Размер:** ~4KB
- **Время чтения:** 5-10 минут

### [MODULES.md](MODULES.md)

Подробное описание каждого модуля. Что делает, какие функции, сколько строк.

- **Для:** Разработчиков
- **Размер:** ~11KB
- **Время чтения:** 15-20 минут

### [DEPENDENCIES.md](DEPENDENCIES.md)

Граф зависимостей модулей. Какие модули зависят друг от друга.

- **Для:** Аналитиков, архитекторов
- **Размер:** ~6KB
- **Время чтения:** 10-15 минут
